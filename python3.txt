# --- СТРОКИ ---

Строка имеет индексируемый тип, можно оратиться к индексу и получить его значение
Пример:
a = 'Hello'
a[1] выведит 'e'

Срезы строк:
a[2:] выведет подстроку со второго символа (включительно) - ('llo')
a[2:4] возьмет подстроку со второго символа и остановится на четвертом
a[2:4:2(это шаг)]

Строки являются неизменяемыми

# так же - ассоциативный массив
# или - хеш-таблица

# cat: кошка
# bat: летучая мышь

# ключ: значение

dictionary = {
  'cat': 'кошка',
  'bat': 'летучая мышь'
}

# print(dictionary)
# print(dictionary['cat'])

countries = {
  'Африка': ['Египет', 'Конго', 'ЮАР'],
  'Азия': ['Китай', 'Таиланд', 'Индонезия']
}

africa = countries['Африка']
print(africa)

africa_key = 'Африка'
print(countries[africa_key])

# Добавление в словарь типа ключ: значение
countries['Европа'] = ['Австрия', 'Испания', 'Италия']
print(countries['Европа'])

africa.append('Эфиопия')
print(africa)
print(countries)
# print(len(africa))
print(len(countries['Африка']))

Словари в Python - неупорядоченные коллекции произвольных объектов с доступом по ключу. Их иногда ещё называют ассоциативными массивами или хеш-таблицами.

Типы данных

Числовые:
целые числа - int
вещественные (с плавающей точкой) - float
логические - bool

Строковые
строки - str

Преобразование типов:
int(x) - преобразование к целому числу
float(x) - преобразование к числу с плавающей точкой

Определение типа:
type() - определяет тип данных

if type(x) == int: # int, str, bool, float
    print("ok")
else:
    print("не ок")
